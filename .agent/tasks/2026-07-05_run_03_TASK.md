accuracy. Three data points to compare:

   (`WEB_BUNDLE_STATS.trackedCompanies`).

   (line 117), "40 companies is a curated sample" (line
   143), "40 curated companies" (line 309).

**Neither 8 nor 35 matches 40.** The wording is therefore
inaccurate.

**Which of {8, 35} is the methodology claim actually
about?** Reading the surrounding paragraphs:

- **Line 117-129** frames the "curated set" as the source
  behind "the live home page currently serves the May 2026
  manually-curated snapshot" — i.e. the **bundle**, whose
  unique-company count is 35.
- **Line 143-147** frames "40 companies is a curated
  sample" as "the curated companies we follow" — again the
  bundle-behind-the-served-content, not the automated
  registry.
- **Line 309** frames "40 curated companies" as "our
  corpus" — again the bundle.

All three "40" mentions refer to the same thing: **the
size of the curated-company set behind the served
bundle**, which is currently **35**. The "40" was
approximately-rounded up (35 → 40 is close-but-inflated)
and is now stale enough to be a credibility risk.

**Verdict**: **stale/misleading**. Small copy fix
warranted. Wire all three sites through
`WEB_BUNDLE_STATS.trackedCompanies` for future consistency
+ P1.7b/P1.8a pattern parity. The forbidden list keeps
the helper file itself untouched; import only.
