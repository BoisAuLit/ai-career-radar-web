## Integration contract

After the report regression harness saves `report.md` to scratchpad
in a given run, invoke:

```
node scripts/quote-integrity-check.mjs \
  --report <scratchReportPath> \
  --corpus src/data/web_bundle.json \
  --out    <runDir>/quote_integrity_summary.json \
  --fixture <fixtureId> \
  --source-run-id <runId>
```

where `<runDir>` is the already-existing `.agent/regression_runs/<runId>/`
directory.

## Failure handling

- Missing report → `quote_integrity.verdict = "blocked_no_report"` ·
  `quote_integrity_checker_executed.pass = false`.
- Missing corpus → `quote_integrity.verdict = "blocked_no_corpus"` ·
  `quote_integrity_checker_executed.pass = false`.
- Checker exits non-zero → `quote_integrity.verdict =
  "checker_error"` · store stderr excerpt (capped to ~500 chars).
- Do NOT silently pass as GREEN.
- Do NOT throw and abort the harness — this is telemetry-only.
