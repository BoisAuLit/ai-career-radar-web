## Objective

Design — without implementing — the **Codex
read-only daily planner**: the first
automation-side actor in the AgentOps-3
system pinned by the AgentOps-3a memo
(`.agent/design_memos/2026-07-08_AgentOps-3a_automation_advancement.md`).
The memo specifies:

- **what the planner reads** and what it
  must ignore,
- **what commands** it may execute (all
  read-only) and which it may not,
- **what output** it produces and where
  that output will eventually land,
- **how it classifies risk**, recommends
  a single next task, integrates with
  the still-future report regression
  harness, and interacts with Claude
  Code,
- **how the human + ChatGPT daily loop
  stays ≤ 15 minutes**.

No planner code, no runner, no daemon,
no benchmark fixtures, no `.agent/scripts/**`
edit, no `src/**` edit are produced by
this loop.
